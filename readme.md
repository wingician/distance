1.问题：
从公司出发依次拜访N个客户。已经知晓到每个客户的距离，以及每个客户之间的距离。
求，输入一个总里程，选择一个接近里程的线路，拜访N个客户中的一部分或者全部，然后返回公司。
线路取决于里程总数，拜访的客户个数没有要求。
